# 🧠 Lung Cancer Classification App

A deep learning-based application that detects and classifies lung cancer from medical images using Convolutional Neural Networks (CNN). This project aims to assist in early diagnosis and improve healthcare outcomes.

---

## 📌 Overview

Lung cancer is one of the leading causes of death worldwide. Early detection can significantly improve survival rates. This project uses AI and computer vision to classify lung images into different categories such as normal, benign, or malignant.

---

## ✨ Features

- 🧠 Deep Learning-based image classification  
- 🏥 Medical image analysis  
- 📊 Multi-class prediction (Normal / Cancer types)  
- ⚡ Fast and simple interface  
- 📁 Upload image and get instant results  
- 📈 Confidence score output  

---

## 🛠️ Tech Stack

- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy, Pandas  
- Matplotlib  

---

## 🧪 How It Works

1. Upload a lung scan image  
2. Image preprocessing (resize, normalization)  
3. Model extracts features using CNN  
4. Prediction is generated  
5. Result displayed with confidence

## ⚙️ Installation

### 1. Clone the repository
```bash
git clone https://github.com/rudrava12/lung-cancer-classification-app.git
cd lung-cancer-classification-app
```

### 2. Create virtual environment
```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the app
```bash
python app.py
```
